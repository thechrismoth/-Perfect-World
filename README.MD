# Калькулятор Созвездий 3x3

Веб-приложение для решения головоломки "Созвездия" на поле 3x3.

## Описание

Это одностраничное веб-приложение помогает найти оптимальную последовательность ходов для преобразования игрового поля 3x3 в желаемое состояние (все черные или все белые созвездия).

## Как использовать

1. **Установите текущее состояние доски**: кликайте на круги с номерами, чтобы переключать их цвет между черным и белым
2. **Выберите цель**:
   - "Решить до ЧЕРНЫХ" - найти последовательность для получения всех черных созвездий
   - "Решить до БЕЛЫХ" - найти последовательность для получения всех белых созвездий
3. **Очистка**: кнопка "Очистить" сбрасывает доску в состояние "все черные"

## Особенности

- **Алгоритм поиска решения**: перебирает все возможные комбинации ходов от 1 до 9 для нахождения минимальной последовательности
- **Универсальная последовательность**: "1 → 9 → 5 → 3 → 7" меняет все созвездия на противоположный цвет
- **Адаптивный дизайн**: работает на компьютерах, планшетах и телефонах
- **Темная тема**: интерфейс в темном стиле

## Правила игры

При нажатии на созвездие:
- Выбранное созвездие меняет цвет на противоположный
- Соседние созвездия (сверху, справа, снизу, слева) также меняют цвет

## Техническая информация

- **Технологии**: HTML, CSS, JavaScript (чистый, без зависимостей)
- **Размер поля**: 3x3 (9 созвездий)
- **Алгоритм**: полный перебор комбинаций
- **Поддержка браузеров**: все современные браузеры

## Запуск приложения

Просто откройте файл `index.html` в любом современном браузере.

## Формат решения

При нахождении решения отображается:
- Последовательность номеров созвездий для нажатия
- Подробное пошаговое описание

Пример решения:

```
Последовательность для получения БЕЛЫХ созвездий:

2 → 5 → 8

Подробно:
1. Нажмите на созвездие 2
2. Нажмите на созвездие 5  
3. Нажмите на созвездие 8
```
